type: Changed
title: Redesign the cloud-setup reconcile prompt
---

The prompt that appears the first time you connect a cloud backend that already
holds contacts — where you keep one copy and replace the other — has been
reworked into a compact, centred dialog. Each side is now its own large,
full-width button carrying the copy it acts on ("Use the Dropbox copy —
34 contacts · 6 folders" vs "Replace Dropbox with this device — 1 contact ·
0 folders"), so the counts that inform the decision sit right on the choice,
led by a cloud glyph. The keep-the-cloud-copy option is highlighted as the
safe default.
