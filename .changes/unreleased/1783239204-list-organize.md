---
type: Added
title: Organise contacts on the List page — drag into folders, multi-select, and Move to folder
---

The **List** page is now a place to _organise_, not just browse. Drag any
contact row and drop it onto a folder section to file it there — the section
lights up as you hover, and the list auto-scrolls when you drag near its top or
bottom edge, so a card low in a long list can be lifted straight into a folder
above.

**Select mode** was reworked so it no longer hijacks the page title. The count,
the exit **✕**, the select-all toggle, and the batch **Copy** / **Export**
actions now live in a floating toolbar that hovers at the bottom of the page —
the same style as the import banner — leaving the **List** heading in place. A
**Ctrl / Cmd-click** on any row drops you straight into select mode with that
card ticked, and once several are selected, **dragging any one of them** moves
the whole selection into a folder at once.

Every contact row also gained a **Move to folder** right-click action: pick the
destination from a folder dropdown (the whole folder tree, indented, plus **No
folder** to un-group). It moves the selection when the row you click is part of
it. The same **Move to folder** action was added to the side menu's contact and
folder right-click menus, so a folder can be nested from the menu as well as by
dragging.
