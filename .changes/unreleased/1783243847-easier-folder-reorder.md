---
type: Changed
title: Easier folder reordering in the side menu
---

Dragging a folder to **reorder** it among its siblings is far more forgiving:
the "drop between two folders" zone now covers the top and bottom **40%** of a
folder row (it was a thin 25% edge), so a folder dropped near another slots
above or below it — showing the insertion line — instead of accidentally
nesting inside it. Only the middle of a row still nests the folder in. As
before, reordering is available when folders are set to **manual** sort
(**Settings → List**); alphabetical sort keeps them in name order.
