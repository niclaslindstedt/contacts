type: Added
title: Favorites — star the contacts you reach for most
---

Contacts can now be **starred as favorites**, and favorites gather on their own
**Favorites page**. Tap the **heart** in a contact card's header — or the heart
on any row of the List overview — to star or unstar a card. The new **Favorites**
button, seated next to List in the sidebar's action grid, opens a page that
mirrors the List overview — the same folder grouping, and the same Select / Copy
/ Export tools — filtered to just the starred contacts (a folder with no
favorites drops out). A card opened from the Favorites page carries a back button
to it, exactly as the List page does. Favorites are a per-contact flag stored in
the document, so they ride along in the JSON backup and sync across your devices.
