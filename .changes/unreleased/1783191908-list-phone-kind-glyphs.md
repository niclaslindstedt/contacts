---
type: Changed
title: Phone types show as glyphs in the List and Favorites
---

When a contact shows more than one phone number in the **List** or **Favorites**
page, each number now leads with a small type **glyph** — a person for
**Private**, a briefcase for **Work** — instead of the type spelled out in text.
It's the same two marks the edit form's type toggle already uses, so the type
reads at a glance while the number itself gets the room. A contact with a single
number carries no marker, as before.
