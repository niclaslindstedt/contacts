---
type: Changed
title: Folder sections on the List page read as collapsible separators
---

On the **List** page, a folder's heading is no longer a foldery row with a
folder icon — it's now a tinted **separator band** with its own colour set apart
from the contact rows, an uppercase label, a disclosure caret, and the member
count. It reads as a divider between groups you can collapse and expand, rather
than a folder to open, while still lighting up in the accent when you drag a
contact onto it.

The List header also gains a **collapse / expand all** button, next to
**Select** — one tap folds every section shut (handy for jumping between groups
in a long list), and once they're all collapsed the same button expands them
all again. It mirrors the folder toggle the side menu's **Contacts** header
already carries.
