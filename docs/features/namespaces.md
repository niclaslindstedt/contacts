# Namespaces

A namespace is a whole separate address book — its own folders, its own
contacts, its own undo history. Use one for personal contacts and one for work,
or one per client.

- Switch from the side menu's header; manage (create, rename, restyle, delete)
  from the cog.
- Drag a contact or a whole folder onto another namespace's row to move it
  across.
- The browser tab's favicon follows the active namespace's glyph, so a glance
  tells you which book you're in.
