# Tags

Tags let you **file a contact under as many labels as you like** — "Boat club",
"Board games", "Book club", "Work trip 2026". Where the single
[relationship](feature:relationship) says how you know someone, tags are an open
set you grow as you go, with no fixed list to pick from.

In edit mode the Tags field shows the card's tags as little chips and a text box
underneath. Start typing and it suggests tags you've already used on other cards,
so the same label stays spelled the same way; press **Enter** to add the one you
typed, tap a chip's **×** to remove it, or hit **Backspace** on the empty box to
lift the last one. Tags are matched case-insensitively, so "Boat Club" and "boat
club" won't pile up as two.
