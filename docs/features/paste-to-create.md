# Paste to create

Adding someone you already have in an email is now a single paste. When you copy
a contact from your mail app or address book you usually get their name and
address together, like `Firstname Lastname <firstname.lastname@gmail.com>`. With
a keyboard attached, **paste that anywhere in the app** — as long as you're not
writing in a field — and a new contact is filed with that name and email, and
their card opens ready to read.

A plain email address on its own works too. Anything that isn't a name-and-email
(a sentence, a phone number, a link) is left alone, so an ordinary paste into a
note or the search box still pastes where you expect.

If you paste something by mistake, the **Undo** on the banner that appears
removes the new contact in one tap. Prefer to type the details yourself? The
**New contact** button in the side menu still opens an empty card as before, and
dragging a `.vcf` file onto the app is the way to bring in a whole address book —
see [importing contacts](import.md).
