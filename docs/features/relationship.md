# Relationship

Every contact can record **how you know them**. Open a card in edit mode and,
under Details, the **Relationship** field offers five ready-made choices —
**Family**, **Partner**, **Friend**, **Colleague**, and **Business** — that read
in whatever language the app is set to.

Need something the list doesn't have? Pick **Add custom…** and type your own —
"Neighbour", "Mentor", "Landlord", anything. A custom relationship you add once
is remembered and offered on every other card, so the second time you reach for
it it's already a tap away. Each card holds a single relationship; for the looser
"file this person under a few things" kind of labelling, reach for
[tags](feature:tags) instead.
